Cuando instalamos el chart fake-gpu-operator:

kubectl label node <node-name> run.ai/simulated-gpu-node-pool=default

helm repo update

Proceso de upgrade:

helm upgrade gpu-operator oci://ghcr.io/run-ai/fake-gpu-operator/fake-gpu-operator -f fake-gpu-operator-values.yaml --namespace fake-gpu-operator

kubectl delete configmaps -n fake-gpu-operator topology-node2 topology-node3 

kubectl delete pods -n fake-gpu-operator -l app=device-plugin

kubectl delete pods -n fake-gpu-operator -l app=status-updater




kubectl delete --all pods -n fake-gpu-operator
